# System Requirements
- Android Studio
- Retrofit 2
- Room persistence library
- LiveData
- ViewModel

# Features

- Query Google Books Api to find books
- Query, Insert, Update and Delete books in the SQLite
- Rate and Favorite book
- Categorize between: reading, already read and want to read
